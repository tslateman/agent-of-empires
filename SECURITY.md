# Security Policy

## Reporting Vulnerabilities

If you discover a security vulnerability in Agent of Empires, please report it responsibly:

1. **Do not** open a public GitHub issue
2. Email the maintainers or use [GitHub's private vulnerability reporting](https://github.com/njbrake/agent-of-empires/security/advisories/new)
3. Include steps to reproduce, affected versions, and potential impact

We aim to respond within 48 hours and release a fix promptly.

## Supported Versions

Security fixes are applied to the latest release only.
